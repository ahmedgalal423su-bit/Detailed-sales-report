تعليمات النشر على Cloudflare Pages من GitHub:

1) ارفع محتويات هذا المجلد إلى مستودع GitHub، وليس ملف ZIP نفسه.
2) في Cloudflare Pages اختر Connect to Git ثم المستودع.
3) Framework preset: None
4) Build command: npm run build
   أو اتركه فارغاً إذا سمح Cloudflare بذلك.
5) Build output directory: public
6) Root directory: /

الموقع داخل public/index.html وملف _redirects داخل public/_redirects.
